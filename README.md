# My Mac

## 系统拓展

- `CCopy` 记录复制历史
- `ClashX` 代理软件
- `CleanMyMac X` 系统清理
- `Easy New File` 右键新建多种类型文件
- `FinderGo Finder` 中直接使用终端打开文件夹
- `Hidden Bar` 菜单栏隐藏部分菜单
- `Istat Menus` 系统信息监控
- `Iterm2` 终端
- `Magnet` 屏幕切分工具
- `Open in Code` Finder 中直接使用 code 打开文件夹
- `ShortCut Bar` 菜单栏固定文件夹/书签
- `The Archiver` 解压文件
